### 冒泡排序

- 冒泡排序只会操作相邻的两个数据
- 每次操作都会对相邻的两个元素进行比较，如果不满足关系就互换
- 一次冒泡至少一个元素移动到它应该在的位置，重复n次就完成n个数据的排序工作
- 冒泡排序是原地排序算法
- 冒泡排序是稳定的排序算法
- 冒泡排序最好情况时间复杂度是 `O(n)`，最坏的情况是 `O(n<sup>2</sup>)`,平均复杂度是`O(n<sup>2</sup>)`
